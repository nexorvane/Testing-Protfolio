# Claymorphic Geometric Portfolio

A high-visual frontend portfolio website built with React, Vite, and Tailwind CSS, featuring claymorphic design elements and geometric abstraction.

## Tech Stack

- **React 19**
- **Vite** (Build tool)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Visual Style

- **Claymorphism**: Soft UI with exaggerated depth, inner highlights, and rounded edges.
- **Geometric Abstraction**: Floating 3D-like primitives and blur effects.
- **Micro-interactions**: Spring-based animations and hover states.

## Setup Instructions

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Folder Structure

- `/src/components`: Reusable UI elements (Hero, About, Projects, etc.)
- `/src/data`: Mock JSON files for content management
- `/src/hooks`: Custom hooks (Theme management)
- `/src/index.css`: Core design system and claymorphism utilities
